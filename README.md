# Image Changing Animation with CSS

This project demonstrates a simple image changing animation using CSS. Images smoothly transition from one to another with a fade effect, creating a visually appealing gallery or slideshow effect.

## Features

- Smooth fade-in and fade-out transitions
- Easy to customize with your own images
- Pure CSS solution, no JavaScript required
